# Clinic Queue Management System
GROUP 1 built a Clinic Management System using a GUI Base Python Application built with python Tkinter used for managing patient queues in clinics and small hospitals in our society.
#Features
Patient Registration: Add patient ID, name, age, and symptoms
Queue Management: View all waiting patients in a table with scrollbar
Call Next Patient: Serve patient in FIFO order
Delete Selected: Remove patient with confirmation popup to prevent mistakes
Search Patient: Find patients by ID instantly
Statistics: View total registered, waiting, and served patients
Input Validation: Prevent empty fields, invalid age, and duplicate IDs
